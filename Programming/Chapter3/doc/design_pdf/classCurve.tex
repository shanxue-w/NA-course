\doxysubsection{Detailed Description}
\subsubsection*{template$<$int N, typename Real = double$>$\newline
class Curve$<$ N, Real $>$}

This class is used to implement the curve interpolation, which can be used to interpolate the curve in 2D, 3D, and the surface of a ball. The curve is represented by a set of points, and the interpolation method is determined by the method parameter. 

The curve can be interpolated by using the Bspline or \mbox{\hyperlink{classPPoly}{PPoly}} method.

Write all points into \[ x = r \sin \theta \cos \phi, \quad y = r \sin \theta \sin \phi, \quad z = r \cos \theta \]

Then use Bspline to interpolate $ \theta $ and $ \phi $. By polar coordinates, we can transform the 3D curve to a 2D curve

We can find a point $(x_0,y_0,z_0)$ not in the cruve, then the point can project the curve to a plane. \[ x_1 = \frac{z x_0 - z_0 x}{z - z_0}, \quad y_1 = \frac{z y_0 - z_0 y}{z - z_0} \]

Use Bspline to interpolate the projected curve.
