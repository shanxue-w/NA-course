Periodic boundary condition.

$ B(x_i) = y_i $ is needed, which gives the first t\+\_\+size equations. For periodic boundary condition, we need to add one additional equation $ B^{\prime} (x_0) = B^{\prime} (x_{t_size - 1}) $.

Finally we have t\+\_\+size + 1 equations. Then use Eigen to solve the linear system.

The complete boundary condition, give one derivative at the beginning point.

The theorem in the text book, special case, for problem C and D.
